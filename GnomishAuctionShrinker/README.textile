GnomishAuctionShrinker creates a simple, compact auction browse view.  GAS is designed to provide an interface similar to Auctioneer's "CompactUI" in a small standalone addon.

h2. Features

* Simple one-line-per-auction interface
* Adds iLvl and Unit Buyout columns, which are also sortable
* Alt-click to buyout an item
* Always-on page nav buttons (instead of only showing when scrolled to the bottom of the list)
