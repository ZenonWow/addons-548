Simple Notepad
=============================================

Simple Notepad allows you to keep unlimited World of Warcraft notes on your computer and in-game. 

Notes are accessable by all of your toons. 

New notes can be copied and pasted from web pages or from existing computer notes. 

Never have to leave the game to access or create notes again.

Drag and Drop Icons to Create Item Links on any Page

Mini Map Button - If unable to hide open SimpleNotepad_Defaults in a text editor and change "false" to "true"
